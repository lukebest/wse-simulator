# 8×8 Mesh 上 2-ary, 6-flat Flattened Butterfly 的 Color 子拓扑分解

> **目标**：在 8×8 物理 2D Mesh（64 PE）上，通过 **5 个 Color** 时分复用实现完整
> 2-ary, 6-flat FB 逻辑拓扑。

---

## 1. 基本定义

### 1.1 节点编号（行优先 8×8）

```
 0  1  2  3  4  5  6  7   ← row 0
 8  9 10 11 12 13 14 15   ← row 1
16 17 18 19 20 21 22 23   ← row 2
24 25 26 27 28 29 30 31   ← row 3
32 33 34 35 36 37 38 39   ← row 4
40 41 42 43 44 45 46 47   ← row 5
48 49 50 51 52 53 54 55   ← row 6
56 57 58 59 60 61 62 63   ← row 7
```

`node = row × 8 + col`

### 1.2 逻辑拓扑：2-ary, 6-flat

- 逻辑坐标：6 维，每维 2 进制 → `node_id = Σ d_i · 2^i`
- 每维逻辑邻居数：**1**（k−1）
- 有向逻辑链路总数：**384**（1×6×64）
- 逻辑直径：**6 跳**（维度序路由）

---

## 2. Color 分解方案

### 2.1 核心结论

| 指标 | 值 |
|------|-----|
| TDM Color 数 C | **5** |
| 物理边负载下界 | 5 |
| 有向逻辑链路 | 384 |

### 2.2 结构规则（k=2 每维 2 节点翻转）

对维度 `d`，源坐标 `(d₅,…,d₀)`，参考位：

```
j = (d+1) mod 6   当 d 为偶数
j = (d-1) mod 6   当 d 为奇数
```

| d 奇偶 | C0 条件 |
|--------|---------|
| 偶 | d_j = 1 |
| 奇 | d_j = 0 |

低 4 维（d=0..3）在局部嵌入上与 4×4 (2,4) 方案一致；**全局 C=5** 因 8×8 mesh 上
跨维物理路径冲突增加，需 compile 阶段贪心着色补全（非简单 2-Color 扩展）。

### 2.3 各 Color 链路分配（compile 贪心着色）

| Color | 有向链路数 |
|-------|----------|
| C0 | 96 |
| C1 | 96 |
| C2 | 64 |
| C3 | 64 |
| C4 | 64 |
| **合计** | **384** |

### 2.4 物理跳距分布

| 物理跳数 | 链路数 |
|---------|--------|
| 1 | 128 |
| 2 | 128 |
| 4 | 128 |

## 4. 路由示例

| 通信 | 维度序路由 | Color |
|------|-----------|-------|
| `0 → 9` | 0→1(C1) → 1→9(C1) | 单 C1 |
| `0 → 63` | 0→1(C1) → 1→3(C3) → 3→7(C3) → 7→15(C1) → 15→31(C3) → 31→63(C3) | 跨 2 Color |
| `0 → 27` | 0→1(C1) → 1→3(C3) → 3→11(C1) → 11→27(C3) | 跨 2 Color |
| `7 → 56` | 7→6(C0) → 6→4(C0) → 4→0(C0) → 0→8(C1) → 8→24(C3) → 24→56(C3) | 跨 3 Color |

## 5. 与 4×4 对比

64 链路/dim × 6 dim = 384 有向链路；C=5（非 4×4 的 C=2）
