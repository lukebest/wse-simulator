# 8×8 Mesh 上 4-ary, 3-flat Flattened Butterfly 的 Color 子拓扑分解

> **目标**：在 8×8 物理 2D Mesh（64 PE）上，通过 **9 个 Color** 时分复用实现完整
> 4-ary, 3-flat FB 逻辑拓扑。

---

## 1. 基本定义

### 1.1 节点编号（行优先 8×8）

```
 0  1  2  3  4  5  6  7   ← row 0
 8  9 10 11 12 13 14 15   ← row 1
16 17 18 19 20 21 22 23   ← row 2
24 25 26 27 28 29 30 31   ← row 3
32 33 34 35 36 37 38 39   ← row 4
40 41 42 43 44 45 46 47   ← row 5
48 49 50 51 52 53 54 55   ← row 6
56 57 58 59 60 61 62 63   ← row 7
```

`node = row × 8 + col`

### 1.2 逻辑拓扑：4-ary, 3-flat

- 逻辑坐标：3 维，每维 4 进制 → `node_id = Σ d_i · 4^i`
- 每维逻辑邻居数：**3**（k−1）
- 有向逻辑链路总数：**576**（3×3×64）
- 逻辑直径：**3 跳**（维度序路由）

---

## 2. Color 分解方案

### 2.1 核心结论

| 指标 | 值 |
|------|-----|
| TDM Color 数 C | **9** |
| 物理边负载下界 | 9 |
| 有向逻辑链路 | 576 |

### 2.2 结构规则（4 节点组内，与 4×4 (4,2) 相同）

每个维度上 16 个 4 节点组 `{n₀,n₁,n₂,n₃}`，12 条有向链路的**逻辑距离—Color 映射**：

| 逻辑距离 | Color（结构类） | 方向特征 |
|---------|----------------|---------|
| 1 | C0 | 邻接双向 |
| 2 | C1 / C2 | 端点出发 / 中间出发 |
| 3 | C3 | 端到端双向 |

8×8 mesh 上三维叠加后物理边负载下界升至 **9**，compile 贪心扩展为 **C=9**（结构 4 类映射到 9 个时隙）。

## 4. 路由示例

| 通信 | 维度序路由 | Color |
|------|-----------|-------|
| `0 → 9` | 0→1(C2) → 1→9(C1) | 跨 2 Color |
| `0 → 63` | 0→3(C4) → 3→15(C4) → 15→63(C1) | 跨 2 Color |
| `0 → 27` | 0→3(C4) → 3→11(C1) → 11→27(C6) | 跨 3 Color |
| `7 → 56` | 7→4(C0) → 4→8(C1) → 8→56(C1) | 跨 2 Color |

## 5. 与 4×4 对比

192 链路/dim × 3 dim = 576 有向链路；C=9（4×4 同 (4,2) 为 C=4）
