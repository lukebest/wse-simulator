切分规则

父结构：16×16 mesh 上 (k=2, n=8) FB；行/列各 4 位地址；父 C=10。
裁剪：丢弃行 14–15 与列 14–15（共 60 节点），保留 196 节点 = 14×14 mesh。
受影响的链路：dim-3 (列 ±8)、dim-7 (行 ±8) 全部受边界约束；短距离 dim 也有部分链路终点落入被裁剪边角而丢弃。
结果：1 400 条有向链路（满 FB 5 096 的 27%），C 从 49 → 10。
