切分方法（结构规则）

每行 8 节点的行内 FB：行内距离 d 的有向链路共 2·(8−d) 条，需要 d 个 Color（按源列号 mod d 分桶）。1+2+…+7 = 28 个行内结构 Color，其中最大同时活跃数 = 4·4 = 16（中间物理边负载下界）。
每列 6 节点的列内 FB：列内距离 d 需要 d 个 Color，列内 Color 总需求 1+…+5 = 15，最大同时活跃数 = 3·3 = 9。
合并下界：行/列物理边互不冲突，下界 = max(16, 9) = 16。greedy 着色把列 FB 的链路填入行 FB 剩余空闲槽位，最终 C=16=k 行。
