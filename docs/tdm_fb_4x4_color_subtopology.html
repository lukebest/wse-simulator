4×4 Mesh 上的 4-ary, 2-flat Flattened Butterfly

物理层：4×4 2D Mesh（16 PE）。逻辑层：4-ary, 2-flat FB（96 有向逻辑链路）。

通过 4 个 Color 时分复用物理链路；每个 Color 是一个静态路由、物理边不冲突的逻辑子拓扑，合并后构成完整 FB。
