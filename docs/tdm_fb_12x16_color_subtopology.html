切分方法（结构规则）

行 FB (K₁₆)：行内距离 d 链路 2·(16−d) 条/行，最大物理边负载 = ⌊16/2⌋·⌈16/2⌉ = 8·8 = 64。
列 FB (K₁₂)：列内距离 d 链路 2·(12−d) 条/列，最大物理边负载 = 6·6 = 36。
C = 下界 = 64。Color 结构上可看作：C0..C15 主要承载行 FB 短距离 + 列 FB 全部，C16..C47 承载行 FB 中距离 (d=3..6)，C48..C63 承载行 FB 长距离 (d=7..15) 与最长列 FB 链路。
