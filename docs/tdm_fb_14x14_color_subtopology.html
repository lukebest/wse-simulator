切分方法（结构规则）

行/列 FB (K₁₄ 各方向)：距离 d 链路 2·(14−d) 条/线，最大物理边负载 = 7·7 = 49。
C = 下界 = 49。结构上行 FB 占 28+ 个 Color、列 FB 占 28+ 个 Color，因负载分布相同，greedy 把列 FB 全部填进行 FB 留出的 49 个槽位，无需新增。
