# 8×8 Mesh 上 8-ary, 2-flat Flattened Butterfly 的 Color 子拓扑分解

> **目标**：在 8×8 物理 2D Mesh（64 PE）上，通过 **16 个 Color** 时分复用实现完整
> 8-ary, 2-flat FB 逻辑拓扑。

---

## 1. 基本定义

### 1.1 节点编号（行优先 8×8）

```
 0  1  2  3  4  5  6  7   ← row 0
 8  9 10 11 12 13 14 15   ← row 1
16 17 18 19 20 21 22 23   ← row 2
24 25 26 27 28 29 30 31   ← row 3
32 33 34 35 36 37 38 39   ← row 4
40 41 42 43 44 45 46 47   ← row 5
48 49 50 51 52 53 54 55   ← row 6
56 57 58 59 60 61 62 63   ← row 7
```

`node = row × 8 + col`

### 1.2 逻辑拓扑：8-ary, 2-flat

- 逻辑坐标：2 维，每维 8 进制 → `node_id = Σ d_i · 8^i`
- 每维逻辑邻居数：**7**（k−1）
- 有向逻辑链路总数：**896**（7×2×64）
- 逻辑直径：**2 跳**（维度序路由）

---

## 2. Color 分解方案

### 2.1 核心结论

| 指标 | 值 |
|------|-----|
| TDM Color 数 C | **16** |
| 物理边负载下界 | 16 |
| 有向逻辑链路 | 896 |

### 2.2 结构规则（8 节点组内，(4,2) 的 4 节点规则推广）

每个维度 16 个 8 节点组 `{n₀,…,n₇}`，56 条有向链路/组。**逻辑距离—Color 概念映射**：

| 逻辑距离 | 结构 Color 类 | 方向特征 |
|---------|-------------|---------|
| 1 | C0 | 邻接（14 条/组） |
| 2 | C1 / C2 | 端点 n₀,n₇ / 中间 |
| 3 | C3 / C4 | 端点 / 中间 |
| 4 | C5 / C6 | 端点 / 中间 |
| 5 | C7 / C8 | 端点 / 中间 |
| 6 | C9 / C10 | 端点 / 中间 |
| 7 | C11 | 端到端双向 |

8×8 mesh 物理负载下界 **16 = k**，compile 贪心着色 **C=16** 达到最优。

## 5. 与 4×4 对比

448 链路/dim × 2 dim = 896 有向链路；C=16=k（4×4 (4,2) 为 C=4=k/2）
